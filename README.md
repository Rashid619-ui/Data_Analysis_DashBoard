# Hyper_Market_Data-Analysis(Interactive DashBoard creation using MS Excel)
## Project Objective

The Hyper Market wants to create an Annual sales report for 2022. so that, the owner of the Hyper Market can understand their customers and grow more sales in 2023 & 2024

## Dataset used
- <a href="https://github.com/Rashid619-ui/Data_Analysis_DashBoard/blob/main/HyperMarket%20Data%20Analysis%20DashBord.xlsx">Dataset</a>

## Question 
- Compare the sales and order using singal chart.
- which month got the higest sales and order?
- who purchased more - Men or Women?
- what are the diffrent order status in 2022?
- Relation between age and gender based on number of order.
- which channel contributing maximum to the sales?
- higest selling category
- percentage of total orderd delivered
